# API Testing Project – JSONPlaceholder

## Overview
This project demonstrates API testing using JSONPlaceholder.

## Tools Used
- Postman
- Excel
- GitHub

## APIs Tested
- GET /posts
- GET /posts/1
- GET /posts/1/comments
- POST /posts
- PUT /posts/1
- PATCH /posts/1
- DELETE /posts/1

## Testing Performed
- Functional testing
- Status code validation
- Response validation
- Negative testing

## Key Observations
- PUT behaves like PATCH
- API accepts incomplete request body

## Note
This is a mock API and does not enforce real-world validation
